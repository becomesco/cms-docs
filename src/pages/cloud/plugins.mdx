# BCMS Plugins

Using the cloud dashboard you are able only to upload or update existing plugin 
with a new bundled plugin. To learn how to write plugins, please refer to 
[Customization Plugins section](/customization/plugins).

---

## Add a plugin

Login to the [BCMS Cloud](https://cloud.thebcms.com) and open an Instance to
which you would like to upload a plugin. When on the Instance screen click on
**Plugins** card. After that, click on **Add new plugin** and here you will be
able to upload a bundled plugin file.

When plugin is added, you will need to wait a little bit for CMS to rebuild
with new configuration.

---

## Edit a plugin

When you are on the Instance screen click on **Plugins** card. Here you will
see the list of all plugins with a plugin name and **Edit** button next to it.
To edit a plugin, click on **Edit** button.

When plugin is edited, you will need to wait a little bit for CMS to rebuild
with new configuration.

---

## Delete a job

When you are on the Instance screen click on **Plugins** card. Here you will
see the list of all plugins with a plugin name and **Edit** button next to it.
To delete a plugin, click on **Edit** button and after that click on **Delete**
button in the bottom left corner of the modal.

When plugin is deleted, you will need to wait a little bit for CMS to rebuild
with new configuration.
